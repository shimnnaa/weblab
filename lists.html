LISTS IN HTML

Orderd List

An ordered list is used when the order of the items matters (e.g., steps in a process). Items are numbered by default.

  1. decimal numbers.
  2. Lowercase Latin letters.
  3. Uppercase Latin letters.
  4. Lowercase Roman numerals.
  5. Upper case Roman numerals.

Unorderd Lists

An unordered list is used when the order doesn't matter. Items are usually marked with bullet points by default.

  • Default (Disc): Solid round bullets (default style).
  • Circle: Hollow circle bullet points.
  • Square: Square-shaped bullets.

Discription list

A discription list is a type of list used to present terms and their corresponding definitions or descriptions. It's often used for glossaries, descriptions, or FAQs.

eg:-
CPU
The central processing unit; the brain of the computer.
RAM
Random Access Memory; temporary memory that helps your computer run applications.
Hard Drive
A storage device used to store data permanently.
